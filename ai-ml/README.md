# AI/ML 🤖

Learn AI and Machine Learning from math foundations to production MLOps.

| 2 — Deep Learning | `03-deep-learning/` | Perceptrons, backprop, CNNs, RNNs, LSTMs, transfer learning | 6 |
| 3 — LLMs & GenAI | `04-llms/` | Embeddings, attention, transformers, fine-tuning, RAG | 6 |

## The 5th file: notebook.ipynb

AI/ML topics that involve math or visual concepts include a `notebook.ipynb` with pre-rendered visualizations. These notebooks use matplotlib/seaborn and are designed to build intuition *before* writing code.
